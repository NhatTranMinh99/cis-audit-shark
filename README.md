Requirement: ```python3```

Install:
```bash
$ git clone https://github.com/NhatTranMinh99/cis-audit-shark.git

$ cd cis-audit-shark

$ python3 -m pip install -r requirements.txt
```

Start audit: ```sudo python3 run.py```
